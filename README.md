# DutchNumberVerbalizer
English: Transforms a number to a written text in the dutch language. E.g. 1 becomes 'one euro'

Deze functie transformeert een getal naar een uitgeschreven stuk tekst, bijv. 1 wordt 'een euro'. 

Deze functie werkt voor alle getallen tussen de 0 - 1.000.000 en kan omgaan met zowel gehele getallen alswel getallen tot en met 2 decimalen. Bijvoorbeeld '1', '0,25' of '10,3'.

Roep de functie aan met getalNaarTekst(getal).
